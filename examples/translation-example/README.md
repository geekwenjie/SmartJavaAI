# 机器翻译示例

```

└── main
```

## 🚀 快速开始

1. 克隆项目到本地：

2. 导入项目至 IntelliJ IDEA。

3. 根据需要修改模型路径（见各 demo 中注释）。

4. 运行对应的 JUnit 测试类方法即可体验各项功能。
